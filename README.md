# **QR-Code Using Python**

## **Description**
Create QR-codes using pyqrcode package from python for different urls. Here is the list all of the files:
~~~
├── QR-Code Test
    ├── GitHub
        ├── gitHub.png
        ├── gitHub.svg
    ├── LinkedIn
        ├── linkedIn.png
        ├── linkedIn.svg
    ├── Linktree
        ├── linktree.png
        ├── linktree.svg
├── main.py
├── README.md
~~~
## **Installation**
~~~bash
# python2 and python3
pip install pyqrcode
pip install pypng
~~~
